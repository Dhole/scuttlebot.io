Following is controlled by type:contact messages that are published on the user's feed.

Follow users

js
bash
sbot.publish({
  type: 'contact',
  contact: feedId,
  following: true 
}, cb)
sbot publish --type contact --contact {feedId} --following

Unfollow users

js
bash
sbot.publish({
  type: 'contact',
  contact: feedId,
  following: false
}, cb)
sbot publish --type contact --contact {feedId} --no-following
